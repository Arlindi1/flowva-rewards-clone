Flowwa Rewards Clone
A React + Supabase rewards hub clone featuring:

Email/password auth + Google OAuth
Rewards hub dashboard
Daily check-in points (Supabase RPC)
Referral link capture + apply (Supabase RPC)
Tool spotlight claim submission with screenshot upload (Supabase Storage)
Tech Stack
React + TypeScript + Vite
TailwindCSS
Supabase (Auth, Postgres, Storage)
Vercel (deployment)
