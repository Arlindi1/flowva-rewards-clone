# Flowwa Rewards Clone

A **React + Supabase** rewards hub clone featuring authentication, user engagement mechanics, and real-world SaaS-style workflows.

---

## Features

- Email/password authentication + Google OAuth
- Rewards hub dashboard
- Daily check-in points (Supabase RPC)
- Referral link capture and application (Supabase RPC)
- Tool spotlight claim submission
- Screenshot uploads (Supabase Storage)

---

## Tech Stack

**Frontend**
- React
- TypeScript
- Vite
- Tailwind CSS

**Backend / Services**
- Supabase (Auth, Postgres, Storage)

**Deployment**
- Vercel
